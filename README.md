# 🕵️‍♂️ SteganoNinja: LSB Steganography Tool

**SteganoNinja** — це простий та ефективний інструмент на базі Python для приховування текстової інформації всередині зображень формату `.png`. Використовує метод **LSB (Least Significant Bit)**, що дозволяє зберігати дані, не змінюючи візуальний вигляд картинки.

---

## 🛠️ Як це працює?
Програма маніпулює найменш значущими бітами кожного пікселя зображення. 

## ✨ Можливості
* **Hide:** Швидке вбудовування тексту в будь-який PNG-файл.
* **Reveal:** Сканування пікселів та вилучення секретних повідомлень.
* **Clean Interface:** Просте меню в терміналі для зручної роботи.

## 🚀 Встановлення та запуск

1. **Клонуйте репозиторій:**

2. Встановіть необхідну бібліотеку:

pip install stegano

Запустіть скрипт:

python stegano_tool.py

⚠️ Важливе зауваження

Соціальні мережі та месенджери (Telegram, Instagram, WhatsApp тощо) зазвичай використовують агресивне стиснення зображень.
Якщо надіслати картинку як звичайне "Фото", прихований вміст буде знищено.
Для успішної передачі секрету надсилайте файл виключно як "Документ" (Send as File) без стиснення.
